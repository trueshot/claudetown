# Claudetown Tools

**Purpose:** Repo interface — describes what claudetown does, surfaces library and tool info.

**Location:** `c:/clients/claudetown/readme.js`

**Usage:**

```bash
node c:/clients/claudetown/readme.js --tools     # List tools
node c:/clients/claudetown/readme.js --library   # Library index
node c:/clients/claudetown/readme.js --help      # Usage help
```
